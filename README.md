# Personal Finance Analytics Dashboard

## Overview
A Power BI dashboard developed to analyze personal income, expenses, savings, and spending patterns.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel

## KPIs
- Total Income
- Total Expenses
- Net Savings
- Savings Rate
- Core Living Costs

## Features
- Category Filter
- Date Filter
- Expense Analysis
- Daily Spending Trend
- Income vs Expense Comparison

## Key Insights
- Expenses exceeded income by ₹1.54K
- Investment accounted for 54.49% of total spending
- Savings Rate was -77%
